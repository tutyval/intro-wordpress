## Description

Hoteli is a responsive, minimal wordpress theme perfect for hotels, lodges & bed and breakfasts. Hoteli can also be used in any hospitality website from, motels, guest house, Inn, B&B and holiday homes and accomodation. The free theme comes with basic features to get you started, upgrade to pro for all the features you need to get your hotel website looking professional.
 
 
A few notes about the sections above:
 
*   Rooms and front page design are a pro feature
*   Dedicated support only available for pro users



## Frequently Asked Questions

* Can I use this on multiple sites?
 
Yes you can

* Where can i get support from

Use the forum section and a volunteer might help, there's no dedicated support for the free version. For one on one support please upgrade to the pro version
