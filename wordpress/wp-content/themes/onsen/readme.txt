== Installation ==

1. Upload the `onsen` folder to the `/wp-content/themes/` directory
Activation and Use
1. Activate the Theme through the 'Themes' menu in WordPress
2. See Appearance -> Theme Options to change theme options

== Installation Slider ==

1. Create two pages with the title, excerpt and featured images
2. Select the page created in section customization (Appearance ->Customize -> Slider)

== Installation Home Page Box ==

1. Create three pages with the title, excerpt and featured images
2. Select the page created in section customization (Appearance ->Customize -> Home Page)
